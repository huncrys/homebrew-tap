# Huncrys Tap

## How do I install these formulae?

`brew install huncrys/tap/<formula>`

Or `brew tap huncrys/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "huncrys/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
